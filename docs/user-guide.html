1. Getting Started

Start the backend: cd backend && uvicorn main:app --host 0.0.0.0 --port 8000
Start the frontend: cd frontend && npm run dev
Open http://localhost:5173 in your browser
Verify the backend: visit http://localhost:8000/docs for Swagger UI
Phone setup: Run npm install on a laptop, then copy node_modules/ to your phone. The Vite dev server works on the phone.

How it works: Themes use CSS custom properties (--t-bg, --t-surface, --t-primary, etc.) applied to :root. All components reference these variables instead of hardcoded colors, so the entire UI updates instantly when you switch themes.

10. Settings & Modes

Standalone Mode (default)

Both Gemini and Claude APIs are active. Full dual-model experience.

Claude Code Mode

Only Gemini API is active. When you select Claude as provider, the app generates structured JSON context that you can feed into a Claude Code session. This saves Anthropic API costs (subscription vs per-token).

Workspace Transfer

Export your entire workspace (projects, files, chat history, personas) as JSON. Import it in the other mode or on another device.

11. Tips & Shortcuts

Cost savings: Use Gemini Flash for routine tasks, Claude Sonnet for quality work, Opus only for complex reasoning.
Extended thinking: Enable the "Think" checkbox for Claude to see its reasoning process. Budget 4K-8K tokens for most tasks.
Agent pipelines: Chain cost-advisor â†’ prompt-engineer â†’ code-reviewer for a complete optimization workflow.
Playbook search: Search "cost" to find optimization recipes, "agent" for autonomous workflows, "reasoning" for metacognition tools.
Theme switching: Click the palette icon in the NavBar to quickly switch themes. Try Deluxe-CRT for a retro terminal feel or Scratch for a whiteboard doodle aesthetic.
Note: API keys are stored in backend/config.py. Set environment variables GEMINI_API_KEY and ANTHROPIC_API_KEY to override. Never commit keys to public repositories.
