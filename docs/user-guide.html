1. Getting Started

Start the backend: cd backend && uvicorn main:app --host 0.0.0.0 --port 8000 --reload
Start the frontend: cd frontend && npm run dev
Open http://localhost:5173 in your browser
Verify the backend: visit http://localhost:8000/docs for Swagger UI
Phone setup (Termux): Run npm install on a laptop, then copy node_modules/ to your phone. The Vite dev server works on the phone. Use the ESBUILD_BINARY_PATH env var if esbuild fails on ARM.

API Keys

Set environment variables before starting the backend, or configure them at runtime via the Settings page:

VariablePurposeRequired
GEMINI_API_KEYGoogle Gemini API access (text, image, video, audio, VOX)Yes
ANTHROPIC_API_KEYAnthropic Claude API access (text, agents)Standalone mode only

8. KG Studio (57 Databases, 10 Tabs)

🔗

Graph-RAG Knowledge Graph Workspace

A full-featured knowledge graph studio supporting 57 SQLite KG databases with 6 schema profiles, live hybrid search, d3-force graph visualization, NetworkX analytics, RAG chat with streaming and source citations, edge browser with edit/delete, and multi-KG cross-search.

10 Tabs

TabWhat It Does
ExploreInteractive d3-force graph visualization. Click nodes to inspect, drag to rearrange, zoom and pan. Filter by node type.
SearchHybrid semantic search using the formula: 0.40 * embedding + 0.45 * BM25 + 0.15 * graph_boost. Returns ranked results with scores. Also supports multi-KG cross-search across multiple databases.
EditCreate, update, and delete nodes and edges. Full CRUD for building KGs manually. Edge browser with pagination, filtering, and inline edit/delete.
IngestAI entity extraction: paste text and the AI extracts entities and relationships to add to the KG automatically.
AnalyticsNetworkX graph algorithms: centrality metrics, community detection, shortest path finding, and structural statistics.
RAG ChatChat with your knowledge graph. Retrieves relevant nodes as context, streams AI responses, and shows source citations.
CompareStructural comparison between two KGs: shared nodes, unique nodes, overlap metrics.
MergeMerge two KGs with union, intersection, or complement operations.
EmbeddingsEmbedding dashboard: view and manage the embedding index, retrain embeddings, check quality metrics.
BatchBulk operations: batch node/edge creation, import/export.

6 Schema Profiles

KG databases are auto-detected into one of 6 schema profiles. The system adapts its queries and display based on the detected schema.

Creating a New KG

Click "New Database" in the sidebar
Name your KG and choose a schema profile
Use the Edit tab to add nodes and edges manually, or the Ingest tab to extract entities from text using AI
Use the Search tab to verify hybrid search is working, then RAG Chat to converse with your graph

17. Tips & Shortcuts

Cost savings: Use Gemini 2.5 Flash Lite ($0.075/1M) for routine tasks, Claude Sonnet for quality work, Opus only for complex reasoning. The /agent command uses Sonnet automatically.
Extended thinking: Enable the "Think" checkbox for Claude to see its reasoning process. Budget 4K-8K tokens for most tasks, up to 32K for complex analysis.
Slash commands: Type /image a robot painting directly in chat to generate images without switching pages. /tts for speech, /video for video.
Agent pipelines: Chain cost-advisor, prompt-engineer, and code-reviewer for a complete optimization workflow on the Agents page.
KG Studio search: The hybrid search formula (0.40 embedding + 0.45 BM25 + 0.15 graph boost) achieves 88.5% recall. Use multi-KG search to search across all 57 databases at once.
Expert Builder: Create a domain expert by selecting a relevant KG database, writing a focused system prompt, and tuning the 4 retrieval weights. The expert will cite KG nodes as sources.
Game Studio: Complete at least 4 interview questions before generating. Use the Chat tab to iterate on mechanics without regenerating the entire game.
VOX voice: Try different voices for different tasks. Sage (wise, thoughtful) for analysis, Fenrir (excitable, bold) for brainstorming, Puck (upbeat) for general use.
Theme switching: Click the palette icon in the NavBar for instant theme changes. Try Deluxe-CRT for retro terminal aesthetics or Scratch for a whiteboard doodle feel.
MCP server: The workspace exposes all 58+ tools via mcp_server.py as JSON-RPC over stdio for integration with Claude Code and other MCP clients.
Mobile-first: All pages use responsive layouts. Games, Experts, and VOX pages use dropdown pickers instead of sidebars. KG Studio uses a collapsible sidebar pattern.
Playbook search: Search "cost" for optimization recipes, "agent" for autonomous workflows, "reasoning" for metacognition tools, "production" for deployment guides.
Security note: API keys entered in Settings are stored in server memory only and not written to disk. Set environment variables (GEMINI_API_KEY, ANTHROPIC_API_KEY) for persistence. CORS is configured for all origins in dev mode. No authentication is enabled — this is designed for local/trusted environments.
